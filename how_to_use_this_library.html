1. Install Font Awesome to your computer: http://fontawesome.io/3.2.1/assets/font-awesome.zip

 

2. Restart Axure to refresh fonts

 

3. Add Font Awesome CSS link (below) to "Web Fonts" section of the "Generate Prototype Files..." dialog in your project

 

    http://netdna.bootstrapcdn.com/font-awesome/3.2.1/css/font-awesome.css

4. Drag and drop FontAwesome widgets from this library. 

 

Note: Adjust style of all icons using the "FontAwesome" custom style or format individual icons.  Font Awesome icons are adjusted using font size and color.
